## 🚀 About

> **Processed 2M+ daily records with automated ETL pipeline and live visualisations**

**COVID-19 Dashboard** is a production-ready **Data Science** project built with **Python, Pandas, Plotly Dash, PostgreSQL**.
Deployed on **Heroku** with full CI/CD pipeline.

```bash

# 2. Install

# 3. Run
python main.py
```
